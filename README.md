# JinYong's JiangHu

Analyse the relation of characters in JinYong's novels.

数据集包括金庸的14部小说（飞雪连天射白鹿，笑书神侠倚碧鸳）以及手动标注的小说人物名列表文件（/input/people_list_name.txt）。
基于此数据集，分析小说人物关系，包括PageRank计算，社区发现等。
